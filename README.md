## The Technologies used in this project are listed below :- 
1. HTML
2. CSS 
3. React js 
4. TailwindCSS

#Setup
1. You need to download Node modules type this command in terminal where you have cloned this repository
   " npm i "
2. To start the server run 
   " npm run dev "
